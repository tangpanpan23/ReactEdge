# 言刃 ReactEdge

**职场沟通风格演示系统** - 看康辉、董卿、韩寒、成铭如何回答职场高频问题！

## 项目简介

这是一个基于Go语言开发的AI职场沟通演示系统，让你看到不同风格的名人面对相同职场问题时会如何回答。通过输入职场高频问题，AI会模仿康辉、董卿、韩寒、成铭等人的风格给出回答，帮助你：

1. **了解不同沟通风格** - 对比专业得体、温婉大气、犀利直接、逻辑严谨的回答特点
2. **学习职场表达技巧** - 从名人的回答中汲取适用于不同场合的沟通策略
3. **场景化问题演示** - 针对述职答辩、分享会提问、争辩冲突等职场高频场景
4. **风格切换参考** - 根据不同情境选择最合适的沟通风格

## 🧠 系统内核：AI风格模仿引擎

系统通过AI深度学习不同名人的表达特点，构建个性化回答生成模型：

**风格分析四维度：**
1. **语言特征** - 词汇选择、句式结构、修辞手法
2. **思维模式** - 逻辑框架、论证方式、视角选取
3. **沟通策略** - 立场表达、冲突处理、人际技巧
4. **个人特色** - 独特标识、风格标签、核心价值观

**回答生成流程：**
1. **问题解析** - 识别问题类型和场景特征
2. **风格匹配** - 根据问题特点选择最适合的沟通风格
3. **内容生成** - 用目标风格生成自然、贴切的回答
4. **风格验证** - 确保回答符合名人的典型表达模式
