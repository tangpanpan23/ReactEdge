# 🎤 AI酷表达实验室 · 言刃 ReactEdge

用3分钟时间，发现你的表达天赋，掌握顶尖人物的表达风格！

## 项目简介

这是一个基于Go语言开发的AI表达训练系统，专注于青少年表达能力培养。通过3分钟的沉浸式挑战，AI会：

1. **AI风格模仿引擎** - 支持康辉、韩寒、董卿、黄执中等顶尖人物风格
2. **生成个性化表达模板** - 基于用户兴趣和目标风格定制表达框架
3. **实时分析表达DNA** - 深度剖析用户的表达特点
4. **提供针对性训练建议** - 持续提升表达能力

## 🎭 AI风格模仿引擎——"言格分析术"

**技术实现**：AI对目标人物（康辉、韩寒、董卿等）海量语料进行多维分析：

| 分析维度 | 康辉（央视型） | 韩寒（犀利型） | 董卿（共情型） | 黄执中（辩论型） |
|---------|---------------|--------------|--------------|-----------------|
| **节奏模式** | 匀速推进，关键词微停顿 | 变速控制，反问处加速 | 留白艺术，情感点放慢 | 三段式推进，结论处凝聚 |
| **词汇特征** | 高频使用"事实上""本质上" | 偏好"其实""说白了" | 善用"我们""可以感受到" | 专业术语+生活隐喻结合 |
| **思维框架** | 国家-社会-个体三层结构 | 现象解构-本质揭露 | 个人故事-普遍情感-价值升华 | 问题重定义-利害分析-选择构建 |
| **反击策略** | 用事实数据构建防御 | 用对方逻辑反制对方 | 用情感共鸣化解对立 | 用框架优势吞噬对方 |
| **独特武器** | 政策文件精准引用 | 文化现象即时类比 | 文学作品恰当呼应 | 认知心理学概念降维 |

**你的风格训练流程**：

1. **诊断**：AI分析你一段即兴发言，输出"你的当前言格报告"
2. **选择目标**："我想在述职答辩中拥有康辉的沉稳权威感"
3. **拆解训练**：
   - **第1-3天**：只练"康辉式停顿"——AI生成文本，标注📌停顿点，你录音对比
   - **第4-7天**：学习"三层结构法"——针对热点话题，用国家-社会-个人框架组织观点
   - **第8-14天**："事实引用"特训——AI模拟质疑，你需在10秒内调取相关数据支撑
4. **融合创作**：AI生成"混合风格"挑战——"请用韩寒的犀利破题，但以董卿的共情收尾"

- **节奏特征** (层次递进/转折对比/直线表达)
- **独特性分数** (与同龄人差异度)
- **个性标签** (游戏思维者/本质追问者等)

基于分析结果提供：
- **优势强化** 巩固已掌握的表达技巧

- **风格融合** 学习其他表达风格的元素
- **场景适应** 不同场合的最优表达策略
