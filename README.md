# 🎤 AI酷表达实验室 · 言刃 ReactEdge

**职场沟通的"防弹衣"** - 述职答辩、分享会刁难、争辩冲突时的快速反应训练！

## 项目简介

这是一个基于Go语言开发的AI职场危机应对训练系统，专门训练面对领导质疑、场下刁难、不友善争辩时的快速反应能力。通过3分钟的沉浸式挑战，AI会帮你：

1. **建立危机反应内核** - 掌握高压力场景下的思考-反应底层逻辑
2. **三大职场防御策略** - 康辉式专业防御、成铭式逻辑反击、韩寒式态度反制
3. **核心危机场景模拟** - 述职答辩、分享会刁难、争辩冲突等职场高频挑战
4. **反应韧性量化评估** - 测量你的危机沟通能力和心理韧性

## 🧠 系统内核：通用的"思考-反应"模型

高质量临场反应遵循底层逻辑：**先处理情绪和信息，再组织语言和风格**。

**内核模型两步法：**
1. **思考过程（稳住内核）** - 快速解析发问者**真实意图**（考察/挑衅/求真）和问题**本质属性**（事实/观点/情绪）
2. **反应过程（风格外显）** - 根据意图判断，从三大风格策略库调用最合适策略回应

## 🎭 三大风格策略库：康辉、成铭与韩寒

| 风格维度 | **康辉式 (标准得体)** | **成铭式 (辩手机制)** | **韩寒式 (犀利风格)** |
|---------|----------------------|----------------------|----------------------|
| **核心理念** | **真诚与专业**<br>零失误、信息准确、逻辑清晰，以稳定控场赢得信任 | **策略与破局**<br>不纠缠表面，通过逻辑拆解掌控对话走向 | **真实与穿透**<br>不回避冲突，用直接回应打破虚伪，追求观点冲击力 |
| **核心策略** | 1. **结构化表达**<br>用"现状-进展-计划"框架<br>2. **缓冲确认**<br>"这个问题很重要，我思考一下"<br>3. **事实筑底**<br>观点均有数据支撑 | 1. **避实就虚**<br>抓住逻辑漏洞反击<br>2. **请君入瓮**<br>连续提问诱导对方<br>3. **归谬反驳**<br>承认后推导荒谬结论 | 1. **反问拆解**<br>质疑问题前提动机<br>2. **极致坦诚**<br>放大"非常规"化解攻击<br>3. **幽默反刺**<br>用比喻回应挑衅 |
| **最佳场景** | 面试、述职、汇报、协调 | 辩论、谈判、应对刁难 | 应对质疑、回击冒犯、打破僵局 |
| **关键话术** | **面试问缺点**：<br>"我正在提升跨部门推动力，通过提前同步甘特图，将协同效率提升20%" | **质疑方案成本**：<br>"关注投资回报率，我的方案虽初始高15%，但自动化降低30%运维成本" | **被嘲讽变了**：<br>"如果狂是为了说违心话，那我怂了。现在觉得做扎实比说漂亮更重要" |

> **风格融合提示**：最高境界是**内核稳定如康辉，逻辑缜密如辩手，关键时刻保有韩寒式的真诚与锋芒**。

**你的风格训练流程**：

1. **诊断**：AI分析你一段即兴发言，输出"你的当前言格报告"
2. **选择目标**："我想在述职答辩中拥有康辉的沉稳权威感"
3. **拆解训练**：
   - **第1-3天**：只练"康辉式停顿"——AI生成文本，标注📌停顿点，你录音对比
   - **第4-7天**：学习"三层结构法"——针对热点话题，用国家-社会-个人框架组织观点
   - **第8-14天**："事实引用"特训——AI模拟质疑，你需在10秒内调取相关数据支撑
4. **融合创作**：AI生成"混合风格"挑战——"请用韩寒的犀利破题，但以董卿的共情收尾"

## 🎯 三大核心危机场景

### 🔥 **述职答辩：面对领导的各种问题**
领导在述职会上抛出的问题往往最具挑战性，既考验专业能力，又涉及政治敏感度。

- **数据质疑类**：康辉式用事实数据构建专业防御
- **方案挑战类**：成铭式避实就虚，抓住领导逻辑漏洞反击
- **个人表现类**：韩寒式真诚分享真实经历，展现反思能力
- **团队协调类**：灵活切换风格，平衡个人贡献与团队协作

### ⚔️ **分享会刁难：场下故意挑衅的发问**
分享会后的Q&A环节往往最危险，场下听众可能出于嫉妒、不同意见或纯粹捣乱而发难。

- **技术质疑类**：康辉式用专业术语和数据回应，建立权威
- **动机攻击类**：韩寒式极致坦诚，放大"非主流"选择化解攻击
- **逻辑陷阱类**：成铭式请君入瓮，诱导发问者进入己方论证范围
- **人身攻击类**：韩寒式幽默反刺，用比喻化解恶意而不失风度

### 💥 **争辩冲突：不友善的突如其来快速反应**
日常讨论中突然爆发的争辩，往往情绪化强，反应时间短，最考验临场应变。

- **观点对立类**：成铭式归谬反驳，承认对方观点后推导荒谬结论
- **情绪爆发类**：韩寒式反问拆解，质疑对方观点的前提和动机
- **逻辑死缠类**：康辉式缓冲确认，用"这个问题很重要，我思考一下"争取时间
- **恶意挑衅类**：韩寒式幽默反刺，用巧妙比喻瓦解对方的攻击性
